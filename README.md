# Low Ammo Clicks

**Low Ammo Clicks** is a mod for **Zumbi Blocks 2** that adds distinct clicking sounds when you are shooting your gun with low ammo.

## Features

- Plays a subtle and distict clicking sound when you're shooting your gun with the ammo in the mag is 30% or lower.
- Easily replace the click sound with your own .mp3 sound file in the mod's `Sounds` folder.

## Installation

1. Make sure you have the latest version of **BepInEx 5** installed.
2. Download the latest version of the mod at the [releases](https://github.com/tbv001/ZB2.LAC/releases) page.
3. Copy the contents of the zip file into the `plugins` folder of your BepInEx installation:

```
Zumbi Blocks 2 Open Alpha\BepInEx\plugins\LowAmmoClicks
```
